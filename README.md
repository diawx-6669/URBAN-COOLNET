# 🌿 Urban CoolNet — Умный Зелёный Каркас (Full Tech Documentation)

**Urban CoolNet** — это эко-инженерная платформа, использующая глубокое обучение (Deep Learning) и геопространственный анализ для решения глобальной проблемы урбанистических «островов тепла» (UHI). Система переводит планирование озеленения из плоскости «интуитивного ландшафтного дизайна» в плоскость математически обоснованного Data-Driven менеджмента.

---

## 🧠 Глубокая архитектура ИИ-модулей

Платформа состоит из трех критических ИИ-узлов, каждый из которых решает свою задачу:

1.  **Гео-сегментация (U-Net + Remote Sensing):**
    * **Вход:** Спутниковые снимки высокого разрешения (Sentinel-2, Landsat 8).
    * **Процесс:** Нейросеть U-Net выделяет на карте асфальтированные покрытия, бетонные массивы и существующие зеленые зоны.
    * **Результат:** Точная тепловая карта города с выделением критических зон перегрева (Hotspots).

2.  **Микроклиматический анализ (Random Forest + TIRS):**
    * **Вход:** Данные тепловых инфракрасных датчиков (TIRS) и слой инфраструктуры OpenStreetMap.
    * **Процесс:** Алгоритм случайного леса анализирует корреляцию между плотностью застройки, альбедо поверхностей и текущей температурой воздуха.
    * **Результат:** Оценка того, как изменение коэффициента озеленения в конкретной точке повлияет на снижение локальной температуры.

3.  **Финансовый ROI-модуль (LightGBM):**
    * **Вход:** Исторические данные о затратах на полив, стоимости деревьев и снижении энергопотребления (кондиционирование зданий).
    * **Процесс:** Градиентный бустинг рассчитывает ожидаемую экономию бюджета города за 5–10 лет.
    * **Результат:** Готовый бизнес-кейс для мэрии: «Где высадка 100 деревьев принесет максимальный экономический эффект».

---

## 🏗 Технологический стек — подробно

* **Инфраструктура хранения:** PostgreSQL с расширением **PostGIS**. Мы храним городскую застройку не как таблицы, а как гео-объекты, что позволяет выполнять сложные запросы типа «найди все парковки в радиусе 500 метров от жилого квартала».
* **API-уровень:** FastAPI обеспечивает асинхронную обработку запросов. Это критически важно, так как расчет «тенистых маршрутов» требует мгновенного отклика при навигации пользователя.
* **Визуализация:** Интеграция с **Mapbox GL**. Мы не просто показываем картинку — мы отрисовываем интерактивные векторные слои (Vector Tiles), которые позволяют фильтровать объекты «на лету» без перезагрузки страницы.

## 🚀 Пользовательские сценарии (User Journeys)

### Для Мэрии (B2G Dashboard):
* **Стресс-тестирование климата:** Симуляция сценария «Что будет, если температура поднимется на 3°C?».
* **Оптимизация бюджета:** Система подсвечивает участки, где высадка деревьев максимально эффективно «охладит» соседние жилые массивы.

### Для жителей (Consumer Mobile App):
* **Тенистый навигатор:** Алгоритм поиска маршрута, который минимизирует время нахождения под прямыми солнечными лучами.
* **Эко-мониторинг:** Возможность увидеть в режиме реального времени температуру в разных районах города.

## 🌍 Локализация и масштабируемость

Система разработана с поддержкой i18n «из коробки» (файл `script.js` использует динамические словари). Это позволяет разворачивать Urban CoolNet в любой стране, просто добавив JSON-файл с переводом. Сейчас мы поддерживаем **RU, KZ, EN**.
